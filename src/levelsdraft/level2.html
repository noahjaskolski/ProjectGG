Welcome to Level 2

All files are organized in a hierarchical file structure within Linux with a root directory (/) Each file or directory is uniquely identified.

Directory and Description

/

This is the root directory which contains the directories needed at the top level of the file hierarchy

/bin

This is the directory for the executable files available to all users

/dev

This is the directory for device drivers

/etc

This is for configuration files, disk config files, valid user lists, groups, ethernet, hosts, where to send critical messages

/lib

This is the directory for shared library files

/boot

This contains files for booting your system

/home

This contains the home directory for users and other accounts

/mnt

This is used for mounting other temporary file systems, as an example, cdrom and floppy.

/proc

This is the root directory which contains the all processes with a process number
/tmp

This is the directory for temporary files used between booting system

/usr

This is the directory for shared files, library files that can be used by other users

/var

This is the directory for files of variable length such as log and print files

/sbin

This is the directory for binary files aka executables that are used in system administration

/kernel

This is the directory which contains kernel files which is the main component of Linux OS. It is the interface between the hardware and itss processes

Handy Commands for Navigating Linux File System

  • cat <fileName> -->displays the contents of a file
  • cd <directoryName> -->moves you into a directory contained in the directory that you are currently in
  • cd .. -->moves you up a directory from where you are currently in
  • cd ~ -->moves you to root directory
  • cp <fileName1> <fileName2> -->enables you to copy a file/directory to specific location
  • mv <fileName1> <fileName2> -->enables you to move a file/directory from one location to another OR it can be used to rename a file/directory
  • rm <fileName> ~ -->remove file
  • mkdir <fileName>~ -->makes a new directory
  • rmdir <directoryName>~ -->removes the directory
