    Welcome to Level 4

    Let's learn more about editors. There are a few that our commonly known such as vim, nano, or vi. Text editors enable you to write scripts which you will use in the next level.

    To create files in vim, you will need to enter vim filename.extension so for example: vim myfile.txt. This would create a text file named myfile, but the extension can be used to create text files, bash scripts,

    Practice Time:

    Go to terminal and view test.txt---cat test.txt
    In terminal type sudo vim test.txt
    You are using vim to view test.txt. By using the i command, you move to insert mode. Do this now.
    Now go to the end of the file and type your name and favorite color.
    Press esc to exit insert mode
    Now type :wq to save and quit file
    Let's test your modifications. In terminal, type cat test.txt.
    You should see your addition.
