Important Concepts

To Understand what Bash scripting is, you need to understand some key terms: shell, terminal, the Linux Operating System, and Bash. Keep in mind that Bash scripting is a powerful tool to create "programs", but there are fundamentals to be learned.

Let's start with the shell which is a command interpreter program that takes commands in a terminal and passes the commands to the operating system to perform. A terminal, also known as terminal emulator, is a program that acts like a wrapper for the shell. This can be seen as a GUI (Graphical User Interface) or a CLI (Command Line Interface. This allows you to access the shell via the command line. It is basically the user interface to the computer's operating system. Linux is the open source Unix-like operating system developed by Linus Torvalds. There are many different distributions of Linux which are likened to flavors of the Linux OS with different enhancements and features. Some popular distributions include Ubuntu, Kali, Fedora, Red Hat, etc.

The Bourne shell was written by Stephen Bourne as an upgrade to the original Thompson shell. The term Bash is an acryonym for the Bourne-Again SHell which is a command language interpreter. It came into existence when the GNU project sought to make a free Unix clone operating system combined with the Bourne shell.

Bash scripting is a text file that contains a series of commands that is to run tasks automatically when executed. It can save a tremendous amount of type or remove user error when entering commands in the terminal. We will learn more about this later.

Okay enough history, in this level you will learn about several commands that are helpful.

Commands:

  • date-->provides today's date
  • pwd-->shows you the power working directory which is where you are in file system
  • ls-->lists all the files based on where you are at in the file structure
  • ls -l -->provides additional information for each of the files, such as file or directory
  • man cat-->this shows the cat manual page. The cat command enables you to concatentate files and print this to standard output. The man command can be used to be the manual page for any program. Try man cut or man grep...

Now, in the terminal start by typing each of the commands and observe the output.
