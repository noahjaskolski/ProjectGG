We have covered many topics in this tutorial. We have learned what a bash shell is, useful commands to use in the terminal, the file hierarchy within Linux, categories of users and their permissions, and TOBEADDED. Now, let's make a simple bash script.

Structure of a Bash script

Here is a sample of a bash script that you are familiar with:
#!/bin/bash
echo "Hello World"

The symbols: #! are known as a shebang and it is the start of the script file as an executable for the loader to execute the script with the path/to/script so in this case it loads the path to bin using script bash.

Bash scripts are like a minature program that is created to run specific commands in the terminal. There are many uses of bash scripts that can save time and deliver consistent results.

We are going to create a bash script to greet a user with the date and then add any two numbers. First, let's create a file.In the terminal, enter sudo vim myProgram.sh (this allows you to open a text editor called vim and named myProgram.sh). Now you will see a blank editor, to edit the file, you will need to go into insert mode by typing i." Type the shebang #!/bin/bash
Then we need to add our commands to run in the bash script:

echo "Enter your name"
read name
echo "Welcome $name. Today," `date` "is a good day!"
echo "Enter your first number"
read x
echo "Enter your second number"
read y
((sum =x+y))
echo "Okay, your total is $sum"

You will need to save the file by exiting insert mode and typing :wq. This command tells the editor to save and quit the file. If you are unsure if your file is accurate, you can concatenate a file that we have already created by typing cat testmyprogram.sh. **Note, here we are using the input variable name in the echo command with $name. Also note that the `date` is telling the script to run the command with the symbol ` for the command date. This can be used to execute any command that is natively known in the Linus OS.
